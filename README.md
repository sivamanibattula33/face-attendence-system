# SmartFace Anti-Fraud Attendance System

## Overview
SmartFace Anti-Fraud Attendance System is an AI-powered attendance management application that uses facial recognition and anti-spoofing techniques to ensure secure and accurate attendance tracking.

## Features
- Face Recognition-based Attendance System
- Anti-Spoofing Detection using Laplacian Variance Analysis
- Real-Time Face Verification
- Automated Attendance Logging
- Excel-Based Attendance Storage
- User-Friendly Gradio Dashboard

## Technologies Used
- Python
- OpenCV
- Face Recognition
- NumPy
- Pandas
- Gradio

## Working
1. Capture image from webcam.
2. Detect and verify face.
3. Perform anti-spoofing validation.
4. Match face encodings with registered users.
5. Mark attendance automatically.
6. Store attendance records in Excel.
